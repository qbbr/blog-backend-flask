# blog-backend-flask

Blog REST API on Flask

see [blog-frontend-vuejs](https://github.com/qbbr/blog-fontend-vuejs)

## depends

 * [python3](https://www.python.org/)
 * see [requirements.txt](requirements.txt)

## run

```bash
python3 -m venv venv
 . venv/bin/activate
pip install -r requirements.txt
. .env.dev
flask run
```
